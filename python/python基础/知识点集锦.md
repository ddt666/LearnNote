## partial偏函数

```python
from functools import partial

def ab(a,b):
    print(a,b)
    return a+b

# 先把1传给参数a生成一个新的偏函数par_ab，但不执行
par_ab=partial(ab,1)

# 把2传给第二个参数，并执行
print(par_ab(2))
```
